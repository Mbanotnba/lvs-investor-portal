The LVS-250 is a production-grade neural compute engine designed for edge AI inference in defense and aerospace applications. Featuring a 5-chiplet UCIe 2.0 architecture with integrated LPDDR5 memory, the LVS-250 delivers ~230 TOPS of inference performance in a compact 35x35mm package. Purpose-built for ISR, C-UAS, and autonomous systems requiring real-time perception and decision-making at the tactical edge.

Key Specifications

Performance~230 TOPS (INT8), ~115 TFLOPS (FP16)
Architecture5-Chiplet UCIe 2.0 (Neural + Secure + RF + Beam + MRAM)
Compute64K MAC NPU + 8-core ARM A78AE + Quad-core VDSP
MemoryLPDDR5 up to 32GB @ 102.4 GB/s

Quick Start Example

# LVS SDK - Object Detection Pipeline Example import lvs from lvs.vision import Camera, ImageProcessor from lvs.inference import Runtime, Model # Initialize runtime on LVS-250 runtime = lvs.Runtime(device="lvs250") # Load compiled model model = runtime.load_model("yolov8_lvs.compiled") # Configure camera pipeline camera = Camera.open(channel=0, resolution=(1920, 1080), fps=30) processor = ImageProcessor(resize=(640, 640), normalize=True) # Inference loop for frame in camera.stream(): tensor = processor.preprocess(frame) detections = model.infer(tensor) for det in detections: print(f"Class: {det.label}, Conf: {det.confidence:.2f}")
